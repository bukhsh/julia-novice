Text Editor—Option 1, Visual Studio Code

When you're writing code, it's nice to have a text editor that is optimized for writing code, with features like automatic color-coding of key words.

The default text editor on macOS and Linux is usually set to Vim, which is not famous for being intuitive. If you accidentally find yourself stuck in it, hit the Esc key, followed by :+Q+! (colon, lower-case 'q', exclamation mark), then hitting Return to return to the shell.

We will use Visual Studio Code, an editor that is more forgiving to newcomers and is popular in the Julia community.

  • Windows
  • MacOS
  • Linux

First, you will need to install and configure Visual Studio Code:

  1. Install Visual Studio Code for your platform.
  2. Start Visual Studio Code.
  3. Inside Visual Studio Code, go to the Extensions view by clicking View on the top menu bar and then selecting Extensions.
  4. In the Extensions view, search for the term "julia" in the Marketplace search box, then select the Julia extension (julialang.language-julia) and select the Install button. This will add Julia to your Visual Studio Code and automatically link the text editor to your copy of Julia you installed earlier.
  5. Restart Visual Studio Code.

Then, test that everything works:

  1. In Visual Studio Code, create a new file by clicking File on the top menu bar and selecting New File.
  2. Inside the file, type println("Hello World!").
  3. Save your file by clicking File on the top menu bar and selecting Save. Name your file hello.jl.
  4. Run your file by first pressing Ctrl+Shift+P to bring up a command palette for search all commands available in Visual Studio Code. Then type Julia: Execute active file in REPL in the command palette and press Return.
  5. If everything works, you should see a message "Hello World!" appear.
